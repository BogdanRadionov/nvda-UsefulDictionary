# UsefulDictionary, Автор Богдан Радионов.
Addon для программы экранного доступа nvda, позволяющий получить информацию о выбранном слове в онлайн словарях и википедии. Версия 0.5.
Внимание, для использования аддона на момент написания этого текста, требуется использовать последние тестовые сборки nvda. Используемый функционал будет доступен с релиза nvda 2016.2.

На данный момент поддерживает следующие ресурсы:
Словарь - http://dictionarybox.com
Википедию - http://mediawiki.org
Яндекс переводчик - http://translate.yandex.ru

Использование:
После установки дополнения, необходимо выделить слово/фразу и нажать комбинацию клавиш nvda+w, после чего откроется текстовое окно с информацией, которую можно читать стрелками и копировать, как обычный текст.
Смена ресурса происходит при нажатии комбинации клавиш 
nvda+control+shift+w.
Сменить используемый язык для ресурса - комбинация клавиш
nvda+control+w.
При смене языка для википедии, последняя будет выдавать информацию на соответствующем языке.
