# UsefulDictionary, Автор Богдан Радионов.
Addon для программы экранного доступа nvda, позволяющий получить информацию о выбранном слове в онлайн словарях и википедии. Версия 0.8.

На данный момент поддерживает следующие ресурсы:
Словарь - http://dictionarybox.com
Википедию - http://mediawiki.org
Яндекс переводчик - http://translate.yandex.ru
Яндекс словарь - http://dictionary.yandex.ru

Использование:
После установки дополнения, необходимо выделить слово/фразу и нажать комбинацию клавиш:
nvda+w.
после чего откроется текстовое окно с информацией, которую можно читать стрелками и копировать, как обычный текст.
Отправить текст из буфера обмена можно клавишами:
nvda+shift+w.
Смена ресурса происходит при нажатии комбинации клавиш:
nvda+control+shift+w.
Сменить используемый язык для ресурса - комбинация клавиш
nvda+control+w.
При смене языка для википедии, последняя будет выдавать информацию на соответствующем языке.

Чтобы включить proxy, зайдите в меню настроек NVDA, выберите пункт "Usefull Dictionary" и установите флажок "Use proxy".
Внимание!!! На данный момент, работа proxy реализована только для Яндекс переводчика, автоматизация подбора прокси откровенно не доделана. Если функция перестала работать, снимите и заново установите флажок для "Use proxy".
